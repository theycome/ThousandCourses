## Тестовое задание от компании First Mobile
# Приложение для курсов

* Техзадание нахоится в приложенном pdf-файле
* Ссылка на Figma файл с дизайном там же
* Чтение данных производится из локального json файла, т.к. backend недоступен

## Stack
* kotlin
* MVVM architecture
* Compose UI
* Coroutines + Flow
* Multi-module architecture (требование ТЗ)
* Clean architecture

## Libraries used
* Moshi
* Dagger2 + Hilt
* Compose Navigation3
* Spotless
* Arrow
* kotest
